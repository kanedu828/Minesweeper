# Minesweeper
Plays minesweeper!

PlayMinesweeper.java

Executes and runs minesweeper

How to play:
Enter coordinates to reveal a square in the format of "y x". Ex: 3 2
To flag or mark a square, enter "y x f". Ex: 3 2 f
